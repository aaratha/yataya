# Yataya

An app for corrdinating and purchasing low-cost travel and hosting solutions for large groups of people.

This app is intended to handle the selection and arrangement of flights, hotels, and venues for large-scale events such as domestic and international weddings, conferences, festivals, retreats, and more.

## Features

- Invite travelers and log important information via automated email RSVPs
- Submit traveler locations, budget, and potential destinations and venues for selection
- Narrow flights, destinations, and venues based on traveler origins and budget
- Purchase tickets, hotel stays, and venues via live API
